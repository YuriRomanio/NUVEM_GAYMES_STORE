# Meu Catálogo

NUVEM GAYMES STORE é uma loja online fictícia e bem humorada criada para um projeto escolar.

Aqui você encontra os produtos mais absurdos, memes e zueiras do universo dos games, tudo com muito humor e sem levar nada a sério. Perfeito para quem gosta de piada interna, meme gen z e cultura gamer brasileira.

### O que o sistema faz?
- Exibe uma loja virtual com produtos engraçados temáticos de games
- Permite cadastrar novos produtos através do login
- Sistema de autenticação e banco de dados real (cadastro, login e salvamento de dados)
- Design responsivo e com bastante personalidade (ou nem tanta)

### Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Supabase (Banco de dados + Autenticação + SQL)
- GitHub Pages (Hospedagem)
